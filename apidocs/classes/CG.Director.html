CG.Director Class

Extends Class
Defined in: cangaja/control/director.js:1
Module: CG

CG.Director the top instance for CG.Screens, CG.Layers, CG.Sprites and so on in the control hierarchy. Its main purpose is to collect CG.Screens under its hood and support some basic screen fading features.
